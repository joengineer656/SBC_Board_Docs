The following reference tables cover every major hardware attribute of the board. Use these specifications to verify compatibility with your project's power budget, operating system requirements, peripheral devices, and enclosure dimensions before you begin assembly.

## Processor

| Attribute      | Value                        |
|----------------|------------------------------|
| Architecture   | ARM Cortex-A, 64-bit         |
| Cores          | 4-core (quad-core)           |
| Clock Speed    | Up to 1.8 GHz                |
| L2 Cache       | 1 MB                         |

## Memory & Storage

| Attribute      | Value                                          |
|----------------|------------------------------------------------|
| RAM            | 2 GB / 4 GB / 8 GB LPDDR4                      |
| Storage        | microSD card slot (UHS-I), eMMC module connector |
| Bootloader Flash | 16 MB SPI NOR flash                          |

## Connectivity

| Attribute    | Value                                          |
|--------------|------------------------------------------------|
| Ethernet     | Gigabit Ethernet (RJ45)                        |
| Wi-Fi        | 802.11 b/g/n/ac, dual-band (2.4 GHz / 5 GHz) |
| Bluetooth    | Bluetooth 5.0                                  |
| USB          | 2× USB 3.0 Type-A, 1× USB-C (OTG)             |

## Video & Audio

| Attribute | Value                                                |
|-----------|------------------------------------------------------|
| Display   | HDMI 2.0 full-size connector, MIPI DSI connector     |
| Camera    | MIPI CSI-2 connector                                 |
| Audio     | 3.5 mm combo headphone/mic jack, I2S digital audio   |

## GPIO

| Attribute     | Value                                    |
|---------------|------------------------------------------|
| Header        | 40-pin (2×20) standard HAT header        |
| Logic Level   | 3.3 V                                    |
| GPIO Count    | Up to 28 user-accessible GPIO pins       |
| Interfaces    | SPI, I2C, UART, PWM, ADC                 |

## Mechanical

| Attribute         | Value              |
|-------------------|--------------------|
| Board Dimensions  | 85 mm × 56 mm      |
| Weight            | ~45 g              |
| Operating Temp    | 0 °C to 70 °C      |
| Mounting Holes    | 4× M2.5 corner holes |

<Note>
  Specifications may vary slightly between board hardware revisions. Always check the revision identifier printed on the PCB silkscreen (labeled **REV x.x**) and compare it against the release notes for your revision before making decisions about peripheral compatibility or power budgeting.
</Note>
