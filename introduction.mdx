Welcome to the official documentation for **SBC Board** — an open-source single board computer designed for makers, developers, educators, and engineers. Whether you are prototyping an IoT device, building an embedded Linux project, or exploring hardware development, SBC Board gives you a compact, affordable, and fully documented platform to work with. These docs cover everything from first power-on to advanced GPIO programming, OS customization, and community resources.

## What's in the Box

Every SBC Board ships with a standard set of items to help you get started right away. Depending on which kit edition you ordered, contents may vary slightly, but a typical retail package includes:

- **SBC Board** — the main board with all onboard components assembled and tested
- **Quick-start card** — a printed reference pointing to these docs and the community forum
- **Regulatory and safety booklet** — compliance information for your region

The following items are **not included** and must be sourced separately:

- microSD card (16 GB or larger, Class 10 / A1 recommended)
- USB-C power supply (5 V, 3 A minimum)
- HDMI or micro-HDMI cable (depending on board revision)
- USB keyboard and mouse
- Optional: case, heatsink, or cooling fan

<Note>
  SBC Board hardware and software are released under open-source licenses. You can find schematics, PCB layout files, BOM, and mechanical drawings in the [official hardware repository](https://github.com/sbc-board/hardware). The software stack and OS images are published under their respective open-source licenses as described in each repository's `LICENSE` file.
</Note>
